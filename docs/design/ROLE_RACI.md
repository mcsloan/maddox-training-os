# Role RACI

Purpose: capture role responsibilities across conceptual, functional, technical, QA, implementation, and governance phases for Closed-Loop methodology work.

| Role | Primary responsibilities |
| --- | --- |
| Project Owner / Product Manager | Owns product intent, acceptance, prioritization, and user trust. |
| Performance Methodology Designer | Owns training logic, U12 appropriateness, load principles, and athletic validity. |
| Product / UX Designer | Owns screen behavior, athlete/parent clarity, and role-based information architecture. |
| AI-Enhanced UX/UI Designer | Owns how explanations, recommendations, heatmaps, and AI-assisted insights appear safely. |
| Data / Analytics Designer | Owns analytics definitions, planned-vs-actual semantics, load metrics, and parent-readable interpretation. |
| Knowledge Engineer / Ontologist | Owns domain ontology, rule mapping, scoring constraints, and schema semantics. |
| Data Engineer | Owns ingestion pipelines, provenance, source trace, and persistence design. |
| Data Scientist | Owns validation approach, thresholds, normalization, confidence, and analytics. |
| Biomechanics / Research Analyst | Owns open-source/research repository review and literature grounding. |
| AI/ML Engineer | Owns future attribute-extraction and personalization pipelines. |
| MLOps Engineer | Owns versioning, monitoring, drift, deployment governance, and rollback of model/rule versions. |
| Systems Architect | Owns technical integration, stack evolution, scalability, security, and current-app transition. |
| AI/Web Developer | Owns future implementation only after design gates. |
| Agentic QA Engineer | Owns adversarial payloads, safety-failure tests, regression fixtures, and automated validation. |
| Governance / Safety Designer | Owns no silent rewrites, parent approval, child-athlete safety, auditability, and unsafe recommendation suppression. |

## Phase Ownership Notes

- Conceptual phase: Project Owner, Performance Methodology Designer, Product / UX Designer, Data / Analytics Designer, Governance / Safety Designer.
- Functional phase: Product / UX Designer, Performance Methodology Designer, Data / Analytics Designer, Knowledge Engineer, Agentic QA Engineer.
- Technical phase: Systems Architect, Data Engineer, AI/ML Engineer, MLOps Engineer, AI/Web Developer, Knowledge Engineer.
- QA/safety phase: Agentic QA Engineer, Governance / Safety Designer, Performance Methodology Designer, Data Scientist.
- Implementation phase: AI/Web Developer and Systems Architect only after design gates pass.
- Governance phase: Project Owner, Governance / Safety Designer, MLOps Engineer, Data Scientist.
