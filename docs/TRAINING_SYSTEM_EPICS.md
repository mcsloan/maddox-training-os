# Training System Epics

## EPIC - Private PWA / Live Training Mode

- Today's plan
- Start session
- Drill cards
- Checkboxes
- Timers
- Notes
- Completion log
- Add to Home Screen support
- Athlete-safe, kid-friendly language

## EPIC - Parent Dashboard + Exports

- Weekly dashboard
- KPI dashboard
- Camp load/recovery view
- Excel export
- PDF weekly printout
- Rink card export
- Season report

## EPIC - Daily Plan / One Day Truth

- Program > Week > Day > Evidence
- Day combines planned work, sport load, training work, support skill, shooting, bike, mobility, reflection, parent notes, and readiness.
- Today, Calendar, History, Dashboard, KPIs, and Exports are projections from canonical Day evidence.
- No silent blank days.

## EPIC - Training System Details

Every athlete-actionable training item should eventually include:

- exact exercise names
- sets/reps/time/rest
- setup
- execution steps
- cues
- mistakes
- parent cue
- source
- video/instruction status
- logging inputs

## EPIC - Source Video / Instruction Coverage

- Every athlete-actionable item needs inline instructions and/or approved video/source link.
- Raw exercise codes cannot stand alone.
- Add instruction/video coverage audit.
- Mobility needs videos/pictures later.
- Codex must not invent video links.
